# 万歩計タウン 設計書

**プロジェクト名**: 万歩計タウン（Pedometer Town）
**目的**: 実際の歩行データをゲームに変換し、体重・速度を考慮した「移動エネルギー」を蓄電池に蓄え、町を発展させる放置・育成ゲーム
**バージョン**: 0.1 (2026-06-15)
**作成者**: Mari (Hermes Agent)

---

## 1. コアコンセプト

- **現実の歩行 → ゲーム内資源**
  - 歩数 + 移動スピード + 体重 → 「移動エネルギー」を算出
  - エネルギーは蓄電池に蓄積（リアルタイム or 定期同期）
- **蓄電池 → 町発展**
  - エネルギーを使って建物を建設・アップグレード
  - 町が発展するとボーナス（エネルギー効率アップ、自動生産など）が出る
- **放置ゲーム寄り**
  - 歩くだけで自然にエネルギーが溜まる
  - プレイヤーは「どこにエネルギーを使うか」の意思決定を楽しむ

---

## 2. 主要数値

### 2.1 入力値
- **体重** (kg)：ユーザーが設定（初回のみ、いつでも変更可能）
- **歩数**：1日ごと or リアルタイム取得
- **移動スピード** (km/h)：GPSまたは手動入力（平均速度）

### 2.2 移動エネルギーの算出式（仮）

```
移動エネルギー (Wh) = 歩数 × (体重 / 70) × (速度 / 5) × 係数
```

- 基準値：体重70kg、速度5km/h で 1歩 = 約0.01Wh 程度
- 体重が多いほど・速く歩くほど効率が上がる（現実の消費カロリーに近づける）
- 係数は後で調整（0.008〜0.015 の範囲でチューニング）

**1日の上限**
- 最大 5000Wh（重いプレイヤーが長時間歩いた場合の上限）

### 2.3 蓄電池
- 容量：初期 10000Wh
- 町が発展すると容量アップ（例: レベル1で +2000Wh）
- 溢れたエネルギーはロスト（または一部変換）

## 3. ゲームシステム

### 3.1 町の構成
- **区画**：住宅街・商業街・工業街・公園など
- **建物**：各区画に建てられる施設
  - 例：
    - 住宅：人口増加 → エネルギー消費効率アップ
    - 発電所：エネルギー生産ボーナス
    - 工場：エネルギー消費してアイテム生産
    - 公園：住民満足度 → ボーナス

### 3.2 エネルギーの使い道
1. **建設**：新しい建物を建てる
2. **アップグレード**：既存建物を強化（エネルギー効率・容量アップ）
3. **研究**：町全体のボーナスを解放
4. **イベント**：特別な町イベントを発動

### 3.3 進行システム
- **毎日ログイン報酬**：歩数に応じたボーナスエネルギー
- **連続歩行ボーナス**：7日連続で歩くと特別報酬
- **町レベル**：発展度で全体ボーナス
